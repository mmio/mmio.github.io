Pages

The site contains more then 5 webpages, amon them the homepage, which is the default one when you visite this web. The about and documentation pages. The blog page which shows all the blog posts made by the author of this site, individual blogposts and the projects page which contains some projects.

Layouts

There is one main layout called default.html which contains the basic structure of a website, the header, footer and all css. Next, there are 2 more inherited layouts the gallery, which is used with the different projects pages, and post, which is used for an individual post, both of those layouts can be found in the _layouts directory. Maybe and third one.

Variables

Most of those are used in the gallery templatex in the look to aquire all the projects and in the navigation.

Datafiles

Datafiles are used for 2 things. The first is the navigation which contains the name of the page and its relative location. The second one is used with the projects gallery where the image, link and description of every individual project can be found.
